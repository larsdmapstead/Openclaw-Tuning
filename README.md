# Openclaw Tuning Log

A public engineering log of how we tune and improve an [OpenClaw](https://openclaw.ai) AI instance running on an NVIDIA DGX Spark.

Topics covered: agent architecture, cron automation, local model routing, context file design, sub-agent orchestration, and what works (and what doesn't).

**Live blog:** [larsdmapstead.github.io/Openclaw-Tuning](https://larsdmapstead.github.io/Openclaw-Tuning)

No personal content — just the technical config, decisions, and lessons.
